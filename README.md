# 智核 SmartAudit

## 产品简介

智核 SmartAudit 是一款面向企业级用户的智能审核系统，支持灵活配置审核规则，对接第三方权威数据源，实现自动化审核与人工复核相结合的高效工作模式。

**核心应用场景**：教育机构人事审核、HR人力资源校验、金融机构资质审查、政府部门数据核查等。

## 系统功能模块

### 1. 租户管理系统
- 租户资料管理（基本信息、logo、联系方式）
- APPID/APPSECRET 管理（开放平台凭证）
- 租户数据统计（累计项目、审核数据量）

### 2. 账户权限系统
- 多级角色：租户最高管理员 → 普通管理员 → 审核员 → 普通员工
- 精细化权限配置，支持一人多角色
- 个人资料管理（头像、邮箱、手机号修改）
- 登录日志（IP、设备类型、操作系统、浏览器）

### 3. 项目管理系统
- 创建和管理审核项目
- 配置入参数据项（支持字符串、数字、日期、下拉、文件等类型）
- 项目完成度统计（进度、审核详情、工作量分布）

### 4. 规则引擎系统
- 可视化规则配置（条件组、条件项嵌套）
- 支持条件组合方式：全部满足(ALL)、任一满足(ANY)
- 内置运算符：等于、包含、区间、正则、身份证验证、学历验证等
- 规则匹配结果详情展示（命中标记、结论说明）

### 5. 任务审核系统
- **任务分配策略**：轮询、负载均衡、效率优先、随机、手动
- 待审核/已审核数据列表（数量统计）
- 审核执行页面（按FIFO顺序处理）
- **二次校验**：对规则未通过数据进行人工复核
- 任务统计（当前任务、今日/本周/本月/本年完成数）

### 6. 三方接口系统
- 配置接口参数（URL、Token、认证信息）
- 支持类型：身份证验证、学历验证、学位验证、银行四要素、运营商验证
- 接口调用日志（请求/响应、耗时、状态）
- 接口调用统计（成功率、平均响应时间）

### 7. 数据统计系统
- 项目完成度统计
- 审核员工作量统计
- 拒绝原因分布
- 租户整体运营数据

### 8. 系统版本管理（独立部署）
- 当前版本显示
- 版本更新检测（自动/手动）
- 版本升级（支持强制更新）
- 升级记录与回滚

### 9. 备份管理系统（独立部署）
- 备份规则配置（全量/差异/增量）
- 定时自动备份
- 手动执行备份
- 备份记录查看
- 备份恢复

## 部署模式

### SaaS 多租户模式
- 共享资源，平台统一托管
- 数据行级隔离
- 平台统一升级

### 独立部署模式
- 独立资源，专属服务器
- 数据完全隔离
- 租户自主升级

## 技术架构

### 系统架构
```
客户端层: PC Web | Mobile H5 | Mini Program | Native App
    ↓
API网关层: Nginx / API Gateway
    ↓
业务服务层: FastAPI 微服务集群 + Celery 任务队列
    ↓
数据层: PostgreSQL + Redis (缓存/会话/消息队列)
    ↓
第三方: 身份证/学历/学位等验证接口
```

### 技术选型

| 层级 | 技术选型 |
|------|----------|
| 前端 | React 18 + Tailwind CSS + Vite + TypeScript + Zustand + TanStack Query |
| 后端 | Python 3.11 + FastAPI + SQLAlchemy 2.x + Pydantic 2.x + Celery |
| 数据库 | PostgreSQL 15+ |
| 缓存/消息队列 | Redis 7+ (RedisMQ/Celery) |
| 文件存储 | MinIO / S3 / OSS |
| 定时任务 | APScheduler + Celery Beat |

### 快速启动
```bash
# 后端
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

# 前端
cd frontend
npm install
npm run dev
```
