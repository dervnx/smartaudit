智核 SmartAudit

# 需求

## 规则引擎配置
- 支持条件配置
- 支持条件组，子组
- 条件项和组均可删除
- 支持配置多条规则，不同规则可匹配不同场景检测
- 支持数字，省市区，标签，时间，学校，专业等
- 条件可全部满足或者任一条件或者其他组合

## 规则匹配结果
- 规则执行结果，可重新执行
- 原始入参数据显示
- 规则匹配详情，规则组需满足条件，按照组层级显示，同时做命中则标绿打对钩，未命中则标红打叉，组也是一样；
- 规则执行结论区块，是否通过规则的结论，通过标绿，打对钩，未通过标红，打叉；总结未通过原因，建议处理

## 规则结果数据列表
- 大面每行分两块，上面块显示入参数据，后方显示结论，点击展开当前行显示匹配结果详情

# 原型

生成一个【智核】的高保真原型页面，包含：

1. 完整HTML+Tailwind CSS + 基础交互JS
2. 页面结果：简单导航、列表、表单、按钮、弹窗
3. 响应式布局，适配移动端
4. 代码可以直接在浏览器运行，方便导入Axure
