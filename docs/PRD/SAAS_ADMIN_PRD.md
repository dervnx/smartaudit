# SaaS 管理平台需求文档

## 1. 系统概述

SaaS 管理平台是平台运营方使用的管理系统，用于管理所有租户、系统版本、配置和运营统计。

**使用对象**：平台运营方管理员

## 2. 功能模块

```
SaaS 管理平台
├── 平台管理员账户
│   ├── 登录
│   ├── 个人信息管理
│   └── 修改密码
├── 租户管理
│   ├── 租户列表
│   ├── 开通租户
│   ├── 编辑租户
│   ├── 禁用/启用租户
│   └── 租户详情
├── 版本发布管理
│   ├── 版本列表
│   ├── 发布新版本
│   ├── 版本详情
│   ├── 设置强制更新
│   └── 禁用版本
├── 全局配置
│   ├── 系统参数
│   └── 通知配置
└── 运营统计
    ├── 租户统计
    └── 使用统计
```

## 3. 详细功能说明

### 3.1 平台管理员账户

#### 3.1.1 账户说明

**账户特性**：
- 平台超级管理员（SYSTEM_SUPER_ADMIN）：系统内置，不可删除，不可修改角色
- 初始账号：admin/saas_admin（首次登录强制修改密码）

**账户字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| 用户名 | 字符串 | 登录账号，唯一 |
| 密码 | 加密 | SHA256 加密 |
| 姓名 | 字符串 | 真实姓名 |
| 手机号 | 字符串 | 联系电话 |
| 邮箱 | 字符串 | 邮箱地址 |
| 头像 | URL | 头像地址 |
| 状态 | 枚举 | active/inactive |
| 最后登录 | 时间 | 最近登录时间 |

#### 3.1.2 登录

**页面**：/saas/login

**功能**：
- 用户名 + 密码登录
- 图形验证码
- 记住登录状态
- 忘记密码（发送重置链接到邮箱）

**交互**：
1. 输入用户名、密码、验证码
2. 点击登录
3. 验证通过跳转首页
4. 验证失败显示错误信息

#### 3.1.3 修改密码

**页面**：/saas/settings/password

**功能**：
- 输入旧密码
- 输入新密码（8位以上，包含字母和数字）
- 确认新密码
- 提交修改

### 3.2 租户管理

#### 3.2.1 租户列表

**页面**：/saas/tenants

**列表字段**：

| 字段 | 说明 |
|------|------|
| 租户名称 | 企业名称 |
| 租户代码 | 唯一标识 |
| 联系人 | 负责人姓名 |
| 联系邮箱 | 邮箱 |
| 联系电话 | 电话 |
| 状态 | active/inactive |
| 创建时间 | 创建日期 |
| 到期时间 | 服务到期日 |
| 操作 | 查看/编辑/禁用 |

**筛选条件**：
- 状态：全部/正常/已禁用
- 创建时间：日期范围

**分页**：每页 20 条，支持跳转

#### 3.2.2 开通租户

**页面**：/saas/tenants/create

**表单字段**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 租户名称 | 字符串 | ✓ | 2-100字符 |
| 租户代码 | 字符串 | ✓ | 小写字母+数字，3-30字符，唯一 |
| 联系人姓名 | 字符串 | ✓ | 2-50字符 |
| 联系邮箱 | 邮箱 | ✓ | 有效邮箱格式 |
| 联系电话 | 字符串 | ✓ | 手机号格式 |
| 公司地址 | 字符串 | ✗ | 详细地址 |
| 开通套餐 | 选择 | ✓ | 基础版/专业版/企业版 |
| 服务期限 | 日期 | ✓ | 到期日期 |
| 独立部署 | 开关 | ✗ | 是否独立部署 |

**交互**：
1. 填写表单
2. 系统生成 APPID/APPSECRET
3. 点击确认开通
4. 发送开通邮件到联系人邮箱

#### 3.2.3 编辑租户

**页面**：/saas/tenants/{id}/edit

**可编辑字段**：
- 联系人姓名
- 联系邮箱
- 联系电话
- 公司地址
- 服务期限
- 套餐变更

#### 3.2.4 禁用/启用租户

**操作**：
- 禁用：确认提示"禁用后该租户所有用户将无法登录，确定禁用？"
- 禁用后租户数据保留，状态变更为 inactive
- 启用：可直接启用，无确认

### 3.3 版本发布管理

#### 3.3.1 版本列表

**页面**：/saas/versions

**列表字段**：

| 字段 | 说明 |
|------|------|
| 版本号 | 如 1.2.3 |
| 版本代码 | 数字，如 10203 |
| 发布类型 | stable/beta/release |
| 是否强制更新 | 是/否 |
| 状态 | 已发布/已禁用 |
| 发布时间 | 发布日期 |
| 租户升级数 | 已升级到此版本的租户数 |
| 操作 | 查看/编辑/禁用 |

#### 3.3.2 发布新版本

**页面**：/saas/versions/create

**表单字段**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 版本号 | 字符串 | ✓ | 格式：x.y.z |
| 版本代码 | 数字 | ✓ | 用于比较，如 10203 |
| 发布类型 | 选择 | ✓ | stable/beta/release |
| 发布说明 | 文本 | ✓ | Markdown 格式 |
| 下载地址 | URL | ✓ | 安装包地址 |
| 文件校验码 | 字符串 | ✓ | SHA256 |
| 最低兼容版本 | 字符串 | ✗ | 可从此版本升级 |
| 强制更新 | 开关 | ✗ | 开启后必须升级 |
| 启用版本 | 开关 | ✓ | 是否立即启用 |

**交互**：
1. 填写版本信息
2. 上传安装包（或提供下载地址）
3. 计算并填写校验码
4. 点击发布
5. 版本发布后，租户端可见

#### 3.3.3 版本详情

**页面**：/saas/versions/{id}

**信息展示**：
- 版本基本信息
- 发布说明（Markdown 渲染）
- 版本覆盖统计
  - 总租户数
  - 已升级数
  - 待升级数
  - 升级率
- 升级记录列表

**升级记录字段**：

| 字段 | 说明 |
|------|------|
| 租户名称 | 租户名 |
| 从版本 | 原版本 |
| 到版本 | 升级后版本 |
| 状态 | 成功/失败/回滚 |
| 升级时间 | 操作时间 |
| 操作人 | 执行人 |

### 3.4 全局配置

#### 3.4.1 系统参数

**页面**：/saas/config/system

**配置项**：

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 平台名称 | 字符串 | 显示名称 |
| 平台 Logo | 图片 | Logo 地址 |
| 默认套餐 | 选择 | 新租户默认套餐 |
| Token 有效期 | 数字 | 分钟数 |
| 单租户最大用户数 | 数字 | 0 表示不限 |
| 单租户最大项目数 | 数字 | 0 表示不限 |
| 注册验证码有效期 | 数字 | 分钟数 |

#### 3.4.2 通知配置

**页面**：/saas/config/notification

**配置项**：

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 开通通知收件人 | 邮箱列表 | 租户开通时通知 |
| 升级通知收件人 | 邮箱列表 | 租户升级时通知 |
| 问题反馈邮箱 | 邮箱 | 用户反馈接收 |

### 3.5 运营统计

#### 3.5.1 租户统计

**页面**：/saas/statistics/tenants

**统计指标**：

| 指标 | 说明 |
|------|------|
| 总租户数 | 累计开通的租户数 |
| 活跃租户数 | 30天内有登录的租户 |
| 本月新增 | 本月新开通租户 |
| 本月流失 | 本月禁用租户 |

**图表**：
- 月度租户增长趋势（折线图）
- 租户套餐分布（饼图）
- TOP 10 活跃租户（柱状图）

#### 3.5.2 使用统计

**页面**：/saas/statistics/usage

**统计维度**：
- 按租户
- 按月份
- 按功能模块

**指标**：
- 累计审核数据量
- 本月审核数据量
- 日均审核量
- 峰值审核量

## 6. API 接口

### 6.1 认证接口

```
POST   /saas/api/v1/auth/login          # 登录
POST   /saas/api/v1/auth/logout         # 登出
GET    /saas/api/v1/auth/me             # 当前管理员信息
PUT    /saas/api/v1/auth/password       # 修改密码
```

### 6.2 租户管理接口

```
GET    /saas/api/v1/tenants              # 租户列表
POST   /saas/api/v1/tenants              # 开通租户
GET    /saas/api/v1/tenants/{id}         # 租户详情
PUT    /saas/api/v1/tenants/{id}         # 编辑租户
DELETE /saas/api/v1/tenants/{id}         # 删除租户
POST   /saas/api/v1/tenants/{id}/disable # 禁用租户
POST   /saas/api/v1/tenants/{id}/enable  # 启用租户
```

### 6.3 版本管理接口

```
GET    /saas/api/v1/versions             # 版本列表
POST   /saas/api/v1/versions             # 发布版本
GET    /saas/api/v1/versions/{id}        # 版本详情
PUT    /saas/api/v1/versions/{id}        # 编辑版本
POST   /saas/api/v1/versions/{id}/disable # 禁用版本
```

### 6.4 配置接口

```
GET    /saas/api/v1/config               # 获取配置
PUT    /saas/api/v1/config               # 更新配置
```

### 6.5 统计接口

```
GET    /saas/api/v1/statistics/tenants   # 租户统计
GET    /saas/api/v1/statistics/usage     # 使用统计
```
