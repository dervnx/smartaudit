# SmartAudit SaaS 管理平台原型

## 1. 原型概述

本原型展示 SmartAudit SaaS 管理平台的核心功能界面，适用于平台运营方管理租户、发布版本等场景。

---

## 2. 页面清单

| 序号 | 页面名称 | 路由 | 功能说明 |
|------|----------|------|----------|
| 1 | 登录页 | /login | 用户登录 |
| 2 | 工作台 | /dashboard | 运营数据概览 |
| 3 | 租户列表 | /tenants | 租户管理列表 |
| 4 | 租户详情 | /tenants/:id | 租户详情/编辑 |
| 5 | 创建租户 | /tenants/create | 新建租户 |
| 6 | 版本管理 | /versions | 版本列表 |
| 7 | 发布版本 | /versions/release | 发布新版本 |
| 8 | 租户版本 | /tenants/:id/version | 租户版本管理 |
| 9 | 独立部署包 | /tenants/:id/deploy | 生成部署包 |
| 10 | 系统配置 | /settings | 全局配置 |

## 4. 交互说明

### 4.1 主要交互流程

#### 创建租户流程
1. 点击「创建租户」按钮
2. 填写租户基本信息（名称、代码、联系人等）
3. 点击「下一步」
4. 配置租户管理员账户
5. 点击「创建」
6. 系统自动创建租户和初始管理员
7. 返回租户列表

#### 发布版本流程
1. 点击「发布新版本」
2. 填写版本信息（版本号、版本名、说明）
3. 上传安装包
4. 选择升级范围（全部租户或指定租户）
5. 点击「发布」
6. 系统处理并通知相关租户

#### 生成独立部署包流程
1. 进入租户详情页
2. 点击「生成独立部署包」
3. 选择基础版本
4. 配置定制项（Logo、域名等）
5. 点击「生成部署包」
6. 等待打包完成
7. 下载部署包

### 4.2 状态说明

| 状态 | 说明 |
|------|------|
| 正常 | 租户正在正常使用 |
| 已禁用 | 租户被暂停使用，无法登录 |
| 待激活 | 租户创建后未完成激活 |
| 升级中 | 正在执行版本升级 |
