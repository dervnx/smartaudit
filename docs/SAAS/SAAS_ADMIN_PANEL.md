## 3. 组件规范

### 3.1 表格组件

| 属性 | 类型 | 说明 |
|------|------|------|
| columns | Column[] | 列配置 |
| data | any[] | 数据源 |
| loading | boolean | 加载状态 |
| pagination | Pagination | 分页配置 |
| rowKey | string | 行唯一标识 |
| selectable | boolean | 是否可选择 |
| onSelect | Function | 选择回调 |

### 3.2 表单组件

| 组件 | 属性 | 说明 |
|------|------|------|
| Input | type, placeholder, prefix, suffix | 文本输入 |
| Select | options, mode, searchable | 下拉选择 |
| DatePicker | format, showTime, range | 日期选择 |
| Switch | checked, onChange | 开关 |
| Radio | options, mode | 单选 |
| Checkbox | options | 多选 |
| Upload | accept, maxSize, listType | 文件上传 |

### 3.3 通用组件

| 组件 | 说明 |
|------|------|
| PageHeader | 页面标题 + 面包屑 |
| Card | 卡片容器 |
| Statistic | 统计数字展示 |
| Badge | 状态徽章 |
| Modal | 弹窗 |
| Drawer | 抽屉 |
| Toast | 消息提示 |
| Confirm | 确认对话框 |
