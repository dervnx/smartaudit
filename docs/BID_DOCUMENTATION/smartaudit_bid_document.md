## 一、公司与产品简介

### 1.1 公司简介

[公司名称]是一家专注于人工智能技术研发和应用的高新技术企业，致力于为政府机构、企事业单位提供智能化、数据审核类产品和服务。公司拥有成熟的研发团队和完善的售后服务体系，已为多家政府单位和大型企业提供信息化建设解决方案。

### 1.2 产品简介

**智核 SmartAudit** 是一款面向企业级用户的智能审核系统，旨在为具有数据审核、校验、核查需求的机构提供一站式解决方案。系统支持灵活配置审核规则，对接第三方权威数据源，实现自动化审核与人工复核相结合的高效工作模式。

### 1.3 核心价值

| 核心价值 | 说明 |
|----------|------|
| **提升效率** | 自动化规则引擎，大幅减少人工审核工作量 |
| **降低误差** | 标准化审核流程，减少人为判断失误 |
| **全程追溯** | 完善的日志记录，支持审核全流程追溯 |
| **灵活配置** | 可视化规则配置，满足多样化审核需求 |
| **安全可靠** | 软硬件多重安全防护，保障数据安全 |

### 4.3 数据安全设计

- **软删除机制**：所有业务表均有 `is_deleted` 字段，支持数据恢复

- **敏感数据加密**：密码等敏感信息 SHA256 + salt 加密存储

## 六、系统安全

### 6.1 身份认证

| 安全措施 | 说明 |
|----------|------|
| JWT Token | Access Token 30分钟有效期，Refresh Token 7天 |
| Token 刷新 | 支持 Refresh Token 自动刷新 |
| 登录限制 | 连续5次登录失败锁定30分钟 |

### 6.2 数据安全

| 安全措施 | 说明 |
|----------|------|
| 密码加密 | SHA256 + Salt 加密 |
| 敏感数据脱敏 | 身份证号等部分脱敏展示 |
| SQL 注入防护 | 参数化查询 |
| XSS 防护 | 输入输出过滤 |

### 6.3 传输安全

| 安全措施 | 说明 |
|----------|------|
| HTTPS | 全站强制 HTTPS |
| CSRF 防护 | Token 验证 |
| 请求签名 | 关键接口请求签名验证 |

### 6.4 软删除机制

- 所有数据删除采用软删除方式
- `is_deleted = 0` 表示正常数据
- `is_deleted = 1` 表示已删除
- 支持后台管理员恢复误删数据
